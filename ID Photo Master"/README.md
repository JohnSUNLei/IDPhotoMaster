# 证件照大师 (ID Photo Master)

一款专业的证件照拍摄应用，帮助用户拍摄标准的证件照，具有实时姿势引导、语音提示和背景替换功能。

## 功能特性

### 📷 相机功能
- 实时相机预览
- 前后摄像头切换
- 闪光灯控制
- 自动对焦和曝光

### 🎯 姿势引导
- 实时人脸检测
- 姿势状态评估（完美/良好/需要调整）
- 头部倾斜角度检测
- 实时指导信息显示

### 🔊 语音提示
- 中文语音指导
- 姿势调整提示
- 拍摄倒计时
- 功能操作反馈

### 🎨 背景替换
- 多种背景颜色选择（白色、蓝色、红色、灰色）
- 智能人像分割
- 证件照规格裁剪
- 高清照片保存

### 📱 用户界面
- 简洁直观的界面设计
- 实时姿势引导框
- 拍摄倒计时显示
- 照片预览和编辑

## 技术栈

- **SwiftUI** - 用户界面框架
- **AVFoundation** - 相机和音频处理
- **Vision** - 人脸检测和姿势分析
- **CoreImage** - 图像处理和背景替换
- **AVSpeechSynthesizer** - 语音合成

## 权限要求

- **相机权限** - 用于拍摄照片
- **麦克风权限** - 用于语音合成（可选）
- **相册权限** - 用于保存照片

## 使用方法

1. **启动应用** - 授予相机权限
2. **对准脸部** - 将脸部放入引导框内
3. **调整姿势** - 根据语音和视觉提示调整姿势
4. **拍摄照片** - 当姿势完美时点击拍摄按钮
5. **编辑保存** - 选择背景颜色和证件照规格，保存到相册

## 注意事项

1. 确保在光线充足的环境下拍摄
2. 保持头部直立，正对摄像头
3. 拍摄时保持表情自然
4. 建议使用后置摄像头获得更高质量的照片
